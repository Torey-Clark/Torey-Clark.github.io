Japanese‐English Translator

This project came about a year after I took a course in Artificial Intelligence. I don't remember why but I suddenly wanted to create a program to translate between Japanese and English. The translations it creates are not amazing by any stretch of the imagination. I'm also not done with it. Right now, it only translates in one direction. I plan to implement bi‐directional translations and plug it into either a personal app or a webpage with a server. The majority of the program is done through Google Colab. If you have any interest in artificial intelligences, check it out. They have tutorials and resources so you can experiment and try out different types of AIs.

Rainmeter

Rainmeter is really cool program that lets users create a ton of different designs for their computer screens that give a new level of customization that everyone would love. People make skins for it and upload them so anyone can use or improve upon them. Anyone that wants to make their own can look at their tutorials which are honestly good.
