Mods

When it comes to modding Minecraft, there are two different versions you can use: Forge or Fabric. Plenty of mods are available for both version but there are also plenty that are only available for a single version. I have found that most mods that you find in modpacks are usually Forge. Fabric mods are generally more light‐weight and can easily satisfy your needs for pseudo‐vanilla multiplayer friendly servers.

Fabric

Fabric Installer
You can't have Fabric mods without the installer.
Fabric API
A utility library for common functionality for other Fabric mods.
Litematica
A pretty nice mod storing schematics for buildings so you have a 3‐D template when building in another world.
MaLiLib
A library for using Litematica.
AppleSkin
A mostly client‐side mod that shows the effects of different foods on the hunger bar.
Enchantment Descriptions
A self‐explanatory mod that gives information on an enchantment without needing to Google it.
Inventory HUD
This is a pretty cool mod that gives you the option to view your inventory constantly to the side and show status effects on you HUD.
Xaero's World Map
Like I said in the Forge section, Xaero has made some nice mods that I need for navigating my world without a sticky note of coords.
OptiFabric
Allows OptiFine with Fabric
Easier Village Trading
Gives additional shortcuts for villager trades.
Entity Culling
This bad boy improves Minecraft's performance by not rendering entities that are not visible to the player. This is already done by default for entities behind the player but this mod extends it to entities hidden behind walls and so on.

Forge

Xaero's Minimap
This is my preferred waypoint manager for all Minecraft servers.
Xaero's World Map
A necessary pair to enhance Xaero's Minimap.
Mouse Tweaks
When you want to craft quickly, you need this mod.
OptiFine
Everyone needs this for some better rendering and frames.
OptiForge
For those extra frames so you don't feel like moving like molasses, you need this if you want OptiFine with Forge mods.
Shulker Tooltip
God bless the creator of this mod that lets me see the contents of my shulker boxes.
MiniHUD
A pretty nifty mod for getting extra information.
